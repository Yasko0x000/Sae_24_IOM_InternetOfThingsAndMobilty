Les objets connectés doivent remonter des informations. L'objectif est de construire un outil de recueil de données via un serveur web. Le résultat pourra être utilisé dans tous les projets où l'on souhaitera obtenir rapidement une remontée d'information.

Le projet est basé sur une carte mentale et une revue de faisabilité dans laquelle les jalons sont identifiés. Des ressources sous forme de lien web ou fiches ressources sont aussi donnés.

.un raspberry PI4 avec BUSTER
.Une carte Micro-SD de 4 Go ou plus (Classe 10 recommandée)
.Une alimentation USB-C 5V pour le Raspberry Pi
.2 ordinateurs
.Un adaptateur de  carte SD
.Un ESP8266+ cable USB-microUSB
.Une plaque de prototypage +fils Dupond
.Résistance de 4,7kOhms
.Un capteur OneWire DS18B20
